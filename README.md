# Sky Pumpkins Game

## 📌 Overview
Sky Pumpkins is an arcade-style game developed using Unity and C#. The player controls a plane that must survive as long as possible by avoiding and destroying enemy pumpkins while collecting power-ups to increase their score and abilities.

## 🎮 Gameplay Features

### 🕹 Core Mechanics
- Player controls a plane using mouse input (click and hold to move the plane)
- Automatic shooting system
- Survive as long as possible with 3 lives
- Game ends after losing all lives

### 🎯 Scoring System
- Enemy pumpkins display numbers representing point values
- Destroying pumpkins increases the player’s score
- High score tracking system

### ⚡ Power-Ups
- Collect coins to gain special abilities:
  - x2 bullets
  - x3 bullets
  - Shield protection
- Power-up charge bar displayed on screen

### 📋 Game Interface
- Instructions screen for how to play
- High score display
- Start menu to begin gameplay

## 🛠 Technologies Used
- Unity Hub
- C#
- Visual Studio 2022
- Game Development Fundamentals
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Open the project in Unity Hub
2. Load the project
3. Press Play in the Unity Editor
4. Use the mouse (click and hold) to control the plane and survive as long as possible

## 📂 Project Structure
- `Assets/` → Game assets (scripts, prefabs, scenes)
- `Scenes/` → Game scenes
- `Scripts/` → Game logic and mechanics
- `Prefabs/` → Reusable game objects
